## 输出目标

生成一份结构化的交接备注，便于教练之间或教练与医生之间传递客户关键信息。

## 结构要求

```
【客户概况】一句话描述客户当前状态
【核心指标】关键数据点列表
【当前方案】正在执行的饮食/运动/作息方案摘要
【风险事项】需要关注的安全相关事项
【跟进重点】下一步需要重点关注的方向
```

## 长度约束

- 总长度控制在 200-400 字
- 条目化呈现，每条 1-2 行
- 不展开分析过程，只给结论和数据

## 语气

- 专业、简洁、中性
- 面向同行（教练或医生），可使用专业术语
- 避免情绪化表述，保持客观

## 示例

【客户概况】女性，45 岁，减重+控糖目标，已跟单 8 周，进展稳定。

【核心指标】
- 体重：入组 72kg → 当前 68.8kg（-3.2kg）
- 空腹血糖：入组 7.2 → 当前 5.8-6.2
- 运动依从：4 次/周

【当前方案】
- 晚餐：杂粮主食 + 蛋白 + 双份蔬菜
- 运动：有氧 3 次/周 + 力量 1 次/周
- 作息：23:00 前入睡

【风险事项】
- 午餐外食频率高，碳水控制不稳定
- 情绪性进食倾向（工作压力大时）

【跟进重点】
- 午餐场景的碳水控制方案
- 血糖异常峰值的触发场景排查
- 第 3 个月评估是否进入自主管理过渡期
