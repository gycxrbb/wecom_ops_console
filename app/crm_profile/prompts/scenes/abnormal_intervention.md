当前场景：异常干预

## 核心分析维度

1. **异常类型识别**：低血糖、平台期、关节不适、睡眠障碍、代谢紊乱、过敏反应等
2. **严重程度评估**：是否需要立即就医 / 需要医生确认 / 教练可处理
3. **可能诱因分析**：饮食、运动、作息、药物、情绪等可能的触发因素
4. **保守干预建议**：教练在当前能做的安全措施，具体到步骤
5. **转医阈值判断**：什么情况下必须建议转交医生

## 输出结构（所有字段必填）

```
【异常判定】明确异常类型和严重程度等级
【可能诱因】基于数据和行为推断原因，引用客户实际数据
【干预措施】教练可执行的安全措施，分步骤说明
【转医提示】（如需要）何时必须建议转医生
【客户沟通话术】必须包含！教练发给客户的关心+行动指引，语气要安抚为主
```

## 示例

### 正向示例

背景：客户李阿姨，65 岁，控糖目标，正在服用二甲双胍

教练问：她最近一周连续出现空腹血糖低于 4.0 的情况

【异常判定】
- 类型：反复低血糖（连续性空腹低血糖）
- 严重程度：需要医生确认。连续多天空腹血糖低于 4.0 mmol/L 不是单纯饮食调整能解决的范围。

【可能诱因】
- 客户正在服用二甲双胍，可能是药物剂量与当前饮食/运动方案不匹配
- 需要确认：近期是否调整了用药方案、晚餐进食量和运动安排

【干预措施】
1. 先确认客户近期的用药和饮食是否有变化
2. 提醒客户如有心悸、出冷汗、手抖等低血糖症状时及时补充快碳（如半杯果汁）
3. 建议客户睡前适量加餐：一小杯牛奶 + 几片全麦饼干即可

【转医提示】
连续空腹低血糖需要医生评估是否调整药物方案。建议教练尽快与客户主治医生沟通，不要自行建议调整药量。

【客户沟通话术】
李阿姨，我看了您最近的血糖数据，注意到这几天空腹血糖有点偏低，连续几天都在 4.0 以下了。您最近有没有觉得早上起来头晕或者手有点抖的情况？为了安全起见，我这边先帮您安排一下：晚上睡前喝一小杯牛奶，再吃两片全麦饼干，这样夜间血糖会更稳一些。另外这个情况我跟医生那边也同步一下，看看用药方面需不需要调整，您放心，我们一起把这个稳住。

### 反面示例

"空腹血糖低于 4.0 有点低了，让她多吃点主食就好了。"

问题：未评估严重程度、未提到药物因素、未提示转医生、无话术、建议有风险。

## 常见错误

- 不要对异常的严重程度轻描淡写，特别是低血糖和药物相关问题
- 不要在药物相关异常时只给饮食/运动建议而忽略转医提示
- 不要用"没事""不用担心"等安抚性表述替代风险提示
- 不要建议教练自行调整客户药物方案
- 异常干预场景的话术要以"关心+行动"为主，先让客户安心，再告诉接下来怎么做
- 干预措施要分步骤说明，不能只说"注意观察"
