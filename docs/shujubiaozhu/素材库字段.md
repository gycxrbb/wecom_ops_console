字段清单整理如下：

  ---
  必填字段（缺一不可，否则整行跳过）

  ┌──────────────────┬────────────────────────────────────────────────┬──────────────────────────────────────────────┐
  │       字段       │                      说明                      │                    示例值                    │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ title            │ 素材标题                                       │ 血糖标准参考图                               │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ summary          │ 一句话摘要                                     │ 展示空腹和餐后血糖的正常范围及糖尿病诊断标准 │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ status           │ 审核状态，只接受 approved 或 active            │ approved                                     │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ rag_enabled      │ 是否启用 RAG 索引，填 yes                      │ yes                                          │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ content_kind     │ 内容类型：image / video / meme / file          │ image                                        │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ visibility       │ 可见范围：coach_internal（仅教练）/            │ coach_internal                               │
  │                  │ customer_visible（可发客户）                   │                                              │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ safety_level     │ 安全等级：general / medical_review             │ general                                      │
  ├──────────────────┼────────────────────────────────────────────────┼──────────────────────────────────────────────┤
  │ copyright_status │ 版权状态：owned（自有）/ licensed（已授权）/   │ owned                                        │
  │                  │ unknown（未知会跳过）                          │                                              │
  └──────────────────┴────────────────────────────────────────────────┴──────────────────────────────────────────────┘

  另外 source_type 列如果填了，必须是 material，不填也行（默认当 material 处理）。

  ---
  素材匹配字段（三选一，必须能匹配到库里的素材）

  ┌──────────────────────────────┬─────────────────────────┬──────────────────────┐
  │             字段             │          说明           │         示例         │
  ├──────────────────────────────┼─────────────────────────┼──────────────────────┤
  │ material_ref                 │ 直接指定素材 ID，最可靠 │ materials.id=12      │
  ├──────────────────────────────┼─────────────────────────┼──────────────────────┤
  │ file_hash                    │ 按文件 hash 匹配        │ 00c83bfb429a303e     │
  ├──────────────────────────────┼─────────────────────────┼──────────────────────┤
  │ file_name 或 source_filename │ 按原始文件名匹配        │ 雅培血糖仪的价值.mp4 │
  └──────────────────────────────┴─────────────────────────┴──────────────────────┘

  匹配优先级：material_ref > file_hash > file_name。如果三个都匹配不到，这行跳过。

  ---
  语义标注字段（影响检索质量和推荐效果）

  这些字段越完整，检索越精准，质量等级越高（ok > medium > weak）：

  ┌────────────────────┬─────────────────────────────────────────┬──────────────┬──────────────────────────────────┐
  │        字段        │                  说明                   │ 影响质量等级 │               示例               │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ alt_text           │ 图片说明 / 图片内容描述                 │   图片必填   │ 空腹血糖3.9-6.1为正常...         │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ transcript         │ 视频/音频转写文本（和 alt_text 二选一） │      是      │ 这个视频介绍了...                │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ usage_note         │ 使用场景说明                            │      是      │ 血糖异常时向客户解释标准范围     │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ customer_goal      │ 适用目标，多值用 | 分隔                 │      是      │ 血糖管理|体重管理                │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ intervention_scene │ 干预场景，参与检索过滤                  │      是      │ abnormal_intervention|qa_support │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ question_type      │ 问题类型                                │      是      │ data_monitoring                  │
  ├────────────────────┼─────────────────────────────────────────┼──────────────┼──────────────────────────────────┤
  │ tags               │ 自由标签，多值用 | 分隔                 │      是      │ 血糖|标准|参考图                 │
  └────────────────────┴─────────────────────────────────────────┴──────────────┴──────────────────────────────────┘

  质量等级规则：
  - ok：title + summary + (alt_text/transcript) + usage_note + tags 全有
  - medium：有 title + summary + usage_note + tags，缺图片说明/转写
  - weak：只有零散信息

  你的标注工作建议：

  1. 先去素材库把每个素材的 ID 记下来（material_ref=materials.id=XX 最稳）
  2. 每个素材重点写好 summary（摘要）和 usage_note（什么时候用）
  3. 图片必须写 alt_text，视频写 transcript 或简短描述
  4. intervention_scene 用已知的场景 key：meal_checkin, meal_review, obstacle_breaking, qa_support, period_review,
  emotional_support, maintenance, abnormal_intervention
