# 系统总览

欢迎来到企微运营平台。

这套系统主要解决 5 类事情：

1. 看今天要发什么
2. 准备内容和素材
3. 把消息发到群里
4. 把多天运营流程编排起来
5. 管理账号、权限、审批和日志

## 侧边栏导航

当前正式导航分为 4 组：

- 核心业务：看板、发送中心
- 数据管理：群管理、模板中心、素材库、定时任务、飞书文档
- 运营配置：话术管理
- 系统设置：系统教学、发送记录、审批中心、用户管理、权限管理

## 推荐上手顺序

如果你是新运营，建议按下面顺序熟悉系统：

1. 看板
2. 群管理
3. 发送中心
4. 素材库
5. 模板中心
6. 定时任务
7. 发送记录

## 你会最常用到的页面

### 看板

适合每天登录后先看一眼，了解：

- 今日待发送
- 最近编排
- 快速入口

### 发送中心

这是最核心的页面，你可以在这里：

- 立即发送
- 测试群发送
- 添加到队列
- 批量发送
- 创建定时任务
- 生成积分排行消息

### 模板中心

适合做系统化、多天、多节点的运营内容安排。

### 发送记录

如果出现“为什么没发出去”，第一时间来这里查看错误信息。
