# 10 分钟上手

这篇文档适合第一次使用系统的同学。

## 第一步：先看今天要处理什么

进入 **看板**，先关注两个区域：

- 今日待发送
- 最近编排

如果今天有待发送任务，可以直接点击进入定时任务查看详情。

## 第二步：认识目标群

进入 **群管理**。

当前有两种视图：

- 内部群
- 外部群视图

你需要先知道：

- 今天消息发给哪些群
- 这些群是内部群还是外部 CRM 群
- 是否已经存在可用的 webhook / 绑定关系

## 第三步：学会发一条测试消息

进入 **发送中心**，按下面顺序操作：

1. 选择发送群组
2. 选择消息类型
3. 填写内容
4. 点击“预览”
5. 点击“测试群发送”

确认测试效果没有问题后，再进行正式发送。

## 第四步：正式发送

如果只发一条消息：

1. 在发送中心完成内容配置
2. 点击“立即发送”

如果要连续发多条：

1. 配第一条消息
2. 点击“新增消息”
3. 继续配置下一条
4. 最后点击“队列发送”

## 第五步：学会排障

如果发送失败，去 **发送记录**。

这里可以看：

- 请求内容
- 响应内容
- 错误信息
- 失败后重试

## 你现在至少要会的 3 条路径

### 路径 A：测试后再发

发送中心 -> 预览 -> 测试群发送 -> 正式发送

### 路径 B：先建素材再发图

素材库 -> 上传素材 -> 发送中心 -> 选择图片/表情包

### 路径 C：先看失败原因再重试

发送记录 -> 找失败项 -> 展开详情 -> 重试
