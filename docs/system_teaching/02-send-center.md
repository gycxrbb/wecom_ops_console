# 发送中心
![image](https://cdn.mengfugui.com/qiwei/docs/images/2026/04/22/image-f5a0fc3b44cb.png)
发送中心是平台最核心的执行页面。

## 这页能做什么

你可以在这里完成：

- 单条消息发送
- 测试群发送
- 手动队列发送
- 批量发送
- 队列定时发送
- 引用模板内容
- 引用运营编排内容
- 生成积分排行消息

## 页面结构

发送中心主要由 3 部分组成：

1. 左侧：消息内容与发送对象
2. 右侧上方：预览
3. 右侧下方：定时任务配置

## 常用按钮

### 基础发送

- 立即发送
- 测试群发送
- 新增消息
- 队列发送
- 测试队列发送

### 批量模式

- 批量发送

## 消息类型

当前支持的消息类型包括：

- 文本
- Markdown
- 图片
- 表情包
- 图文
- 文件
- 语音
- 模板卡片

## 推荐标准操作

### 单条消息

1. 选择目标群组
2. 选择消息类型
3. 填写或引用内容
4. 点击“预览”
5. 点击“测试群发送”
6. 点击“立即发送”

### 连续多条消息

1. 配置第一条消息
2. 点击“新增消息”
3. 继续配置后续消息
4. 最后点击“队列发送”

### 创建定时任务

1. 在发送中心先把消息内容配好
2. 右侧配置执行时间或 Cron 表达式
3. 选择是否跳过周末、是否需要审批
4. 点击创建定时任务

## 积分排行消息

如果你需要发送积分排行：

1. 进入“积分排行”标签
2. 选择 CRM 群
3. 设置排序指标
4. 选择话术风格
5. 点击“生成排行消息”

如果某些群没有进入发送队列，重点看页面里的：

- 跳过原因
- 未绑定发送群提示
- 生成结果摘要
