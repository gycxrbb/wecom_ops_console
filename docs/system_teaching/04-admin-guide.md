# 管理员手册

这篇文档适合 `admin` 账号使用者。

## 管理员比普通运营多了哪些页面

- 话术管理
- 用户管理
- 权限管理

同时，管理员也可以在需要审批的页面中处理审批。

## 用户管理

用户管理用于创建和维护本地登录账号。

### 常见动作

- 新增用户
- 编辑用户
- 修改显示名
- 修改角色
- 启用 / 停用账号

### 注意

这里创建的是本地账号。  
CRM 成员的权限授权，应去“权限管理”处理。

## 权限管理

权限管理用于给账号分配模块访问权限。

### 当前常见权限

- 消息发送
- 定时任务
- 群管理
- 模板管理
- 运营编排
- 素材管理
- 飞书文档
- 发送记录
- 审批操作

### 常见动作

- 搜索成员
- 勾选权限
- 点击保存

## 话术管理

话术管理主要用于积分运营类话术维护。

### 常见动作

- 切换场景
- 修改模板内容
- 保存

## 审批中心

管理员可对待审批任务执行：

- 同意
- 拒绝

## 系统教学

管理员还可以维护系统教学文档。

建议维护原则：

1. 先保证“新人 10 分钟上手”始终最新
2. 发送中心、模板中心、素材库文档优先维护
3. 有新功能上线时，同步补教学文档
