# 企微运营平台教学文档初稿

> 版本：Draft v0.1  
> 适用对象：新入职运营同学、值班运营、管理员  
> 文档目标：帮助使用者在 30 分钟内建立“这个系统能做什么、该从哪里开始、遇到问题去哪里看”的基本认知。

---

## 1. 这套系统是做什么的

这是一套面向企业微信运营场景的发送与运营编排平台，核心能力可以概括成 5 层：

1. **看见**
   - 看板
   - 群管理
   - 发送记录
   - 审批中心

2. **准备内容**
   - 模板中心
   - 话术管理
   - 素材库

3. **执行发送**
   - 发送中心
   - 测试群发送
   - 批量发送
   - 定时发送

4. **运营编排**
   - 运营主题
   - Day / Stage
   - 节点编辑
   - 发布到群

5. **治理与协作**
   - 用户管理
   - 权限管理
   - 审批中心
   - 个人中心

## 3. 新人建议上手顺序

如果你是新运营，建议按下面顺序上手：

1. 先看 **看板**
2. 再看 **群管理**
3. 然后学会用 **发送中心**
4. 再学 **素材库**
5. 再学 **模板中心**
6. 最后再学 **定时任务 / 审批中心 / 发送记录**

如果你是管理员，额外再看：

1. **用户管理**
2. **权限管理**
3. **话术管理**

## 5. 看板

### 5.1 页面用途

看板用于查看今日发送、近期编排和快速入口。

### 5.2 页面按钮与可点击入口

- **刷新**
- 今日待发送卡片中的任务项  
  点击后跳转到 **定时任务**
- 最近编排卡片中的主题项  
  点击后跳转到 **模板中心**
- 快速入口：
  - **发送消息**
  - **运营编排**
  - **定时任务**
  - **飞书文档**
  - **群管理**
  - **素材库**

### 5.3 新人应该怎么用

- 值班时先看“今日待发送”
- 发现失败率异常时去“发送记录”
- 需要快速开始工作时直接走“快速入口”

## 6. 发送中心

### 6.1 页面用途

发送中心是系统最核心的执行页面，用于：

- 立即发送单条消息
- 测试群发送
- 组合成发送队列
- 批量发送
- 创建定时任务
- 引用模板或运营编排内容
- 生成积分排行消息

### 6.2 页面结构

发送中心主要由 3 块组成：

1. **左侧：消息内容**
2. **右侧：预览**
3. **右侧下方：定时任务配置**

### 6.3 左侧配置栏按钮与控件

#### 发送群组

- **全选 / 清空**
- 群组多选下拉框

#### 引用内容

- 点击“引用内容”区域打开选择器
- 可引用：
  - 模板
  - 运营编排
  - 积分排行内容

#### 消息类型

- **文本**
- **Markdown**
- **图片**
- **表情包**
- **图文**
- **文件**
- **语音**
- **模板卡片**

### 6.4 批量模式按钮

当你从运营编排或积分排行中一次选入多条内容后，会进入批量模式。

#### 批量队列区

- **点击队列项查看详情**
- **关闭全部备注 / 开启全部备注**
- 每条内容支持：
  - **备注**
  - **移除**

#### 批量底部动作

- **批量发送**
- **测试群发送**
- **取消发送**

### 6.5 单条模式按钮

#### 当前消息操作

- **立即发送**
- **测试群发送**

#### 手动队列操作

- **新增消息**
- **队列发送**
- **测试队列发送**
- **取消发送**
- **清空**

#### 队列项操作

- 点击队列项查看详情
- **备注**
- **移除**

### 6.6 预告消息区

- **发送推送预告**
- 可编辑自动生成的预告文本

### 6.7 右侧预览区

- **预览**
- 预览模式切换：
  - **模拟效果**
  - **JSON**

### 6.8 右侧定时任务区

- **周期性 / 一次性**
- **执行时间**
- **Cron 表达式**
- **时区**
- **跳过周末**
- **需要审批后执行**
- **跳过日期**
- **执行预览**
- **创建定时任务 / 队列定时发送**

### 6.9 积分排行标签页已确认按钮

发送中心里的“积分排行”能力目前已确认包含：

- **Top N 上限**
- 排序指标：
  - **当前积分**
  - **本周积分**
  - **本月积分**
- 话术风格：
  - **专业**
  - **鼓励**
  - **竞技**
- **全选**
- **一键绑定发送群**
- 单群绑定发送群下拉
- **生成排行消息**

### 6.10 新人最常用的发送路径

#### 路径 A：手动发送

1. 选择群组
2. 选择消息类型
3. 填写内容
4. 点 **预览**
5. 点 **立即发送**

#### 路径 B：测试后再发

1. 配好内容
2. 点 **测试群发送**
3. 确认效果没问题
4. 再点 **立即发送**

#### 路径 C：做成队列

1. 配第一条消息
2. 点 **新增消息**
3. 配第二条消息
4. 最后点 **队列发送**

## 18. 新运营最常见的 5 条操作路径

### 18.1 发一条测试消息

1. 进入 **发送中心**
2. 选择群组
3. 选择消息类型
4. 填内容
5. 点 **测试群发送**

### 18.2 正式发一条消息

1. 在发送中心完成配置
2. 点 **预览**
3. 点 **立即发送**

### 18.3 先建素材再发图

1. 进入 **素材库**
2. 上传图片
3. 回到 **发送中心**
4. 选择图片消息并引用素材

### 18.4 做一个多天运营主题

1. 进入 **模板中心**
2. 点 **新建运营主题**
3. 新增 Day / Stage
4. 新增节点
5. 保存节点内容
6. 点 **发送到群**

### 18.5 看失败原因并重试

1. 进入 **发送记录**
2. 找到失败项
3. 展开看错误信息
4. 点 **重试**

## 19. 管理员额外职责

管理员除了会用运营功能，还要会下面 4 件事：

1. 在 **用户管理** 中创建本地账号
2. 在 **权限管理** 中给账号开模块权限
3. 在 **话术管理** 中维护积分运营话术
4. 在 **审批中心** 中处理需审批任务
